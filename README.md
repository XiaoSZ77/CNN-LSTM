# CNN-LSTM Prediction Results for LEO SCB

This repository provides the prediction results of the CNN-LSTM model proposed in the paper:

**"Low Earth Orbit Satellite Clock Bias Prediction Using the CNN-LSTM Neural Network Model"**

These results are shared to support the reproducibility and verification of the reported findings.


##  Contents

The repository contains:

- Predicted residual series generated by the CNN-LSTM model


##  Data Description

- **Satellites**: GRACE-C, GRACE-D  
- **Prediction arcs**: 120 minutes  
- **Unit**: nanoseconds (ns)
